ERP & POS System (Own Project)

A full-stack enterprise system designed and developed independently, combining ERP and POS functionalities to manage business operations efficiently. Currently in progress.

Tech Stack: Java Spring Boot, Angular, PostgreSQL

Key features implemented:
  • Dynamic menu system for modular navigation
  • Customer and vendor management setup
  • Stock setup and inventory management
  • Stock receive functionality
  • Geographical setup: Country, State, City, and Township management
  • Payment term configuration
  • User and role management system
